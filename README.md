# CampusBox - Student Subscription Box Service

## 📌 About

**CampusBox** is a Pakistan-based monthly subscription box service created specifically for university students. Each box contains a curated collection of stationery, snacks, self-care products, and motivational cards, delivered to subscribers every month.

---

## 📦 Box Tiers

| Tier | Contents | Price |
|------|----------|-------|
| **Basic** | 1 notebook, 1 pen set, 1 snack, 1 motivational card | Rs. 700 |
| **Standard** | Notebook, pen set, 2 snacks, self-care item, motivational card, sticker sheet | Rs. 1,000 |
| **Premium** | Full Standard + bonus item, aesthetic packaging, handwritten note | Rs. 1,400 |

---

## 🎯 Target Audience

- University students (18-26 years)
- Parents, friends, partners (gift subscriptions)
- Institutional clients (university societies, events)
